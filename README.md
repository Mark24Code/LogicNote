# Mark24Note

## Logic NoteBook

逻辑笔记本

不借助软件开发，但是使用 markdown 作为一种约定的数据结构。

* 存储：文件系统、github、git
* 预览：vscode、sublime 预览 markdown； 浏览器预览html
* 读写：文件系统，基于markdown
* 搜索：vscode、sublime 基于本地检索
* 额外功能：Ruby Rake 辅助
  * 同步： git
  * 剪藏：ruby爬虫获得页面
  * 构建：ruby把页面构建成 html
  * 压缩：归档
  * 分析索引


## 约定

# _core/ ruby内核脚本 
# notes/ 笔记的存储之处
